# Java Programming Basics

A well-structured Java repository covering Core Java, Object-Oriented Programming (OOP), Collections Framework, Exception Handling, Multithreading, Generics, File Handling, and college practical programs.

This repository is intended for beginners, students, and anyone looking to strengthen Java fundamentals through organized examples and practical implementations.

## Getting Started

Clone the repository:

```bash
git clone https://github.com/<your-username>/java-programming-basics.git
```

Compile a program:

```bash
javac FileName.java
```

Run the program:

```bash
java FileName
```
